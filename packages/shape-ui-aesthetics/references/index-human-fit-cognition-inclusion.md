# Human Fit, Cognition & Inclusion Index

**Decision question:** Does the aesthetic proposition fit learned expectations, goal models, attention limits, bodies, abilities, contexts, dignity, and belonging?

**Material signals:** A novel direction contradicts learned behavior; users overlook something expected elsewhere; the redesign cuts off a familiar model; one assumed path excludes participation; or an effect layer, color channel, semantic representation, focus path, hit-test layer, or fallback excludes perception or operation.

**Non-signals:** Users understand the model and the issue is preference alone; or a raw contrast token has a known direct correction with no unresolved effect-layer, semantic, focus, operation, or redundant-channel decision.

**Selection seam:** Route visual-effect accessibility for semantic layering, readability, focus, hit testing, redundant channels, and effect fallback. Signal Motion when the independent question is temporal comfort, flashing, interruption, auto-advance, or motion/device degradation; load both only when both decisions are material.

**Cross-domain handoffs:** Signal Interaction when the conceptual model or feedback loop is wrong; Information when goal filtering hides content; Motion when migration or state change needs continuity; Medium when input/output excludes; Meaning when identity or cultural position requires human judgment.
