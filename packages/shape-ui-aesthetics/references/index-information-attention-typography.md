# Information, Attention & Typography Index

**Decision question:** Does perceptual emphasis match task consequence while preserving readable, comparable, and memorable information?

**Material signals:** Attention lands on the wrong element; several elements demand equal urgency; grouping implies false relationships; a critical state is visually inhibited.

**Non-signals:** Users notice and interpret the information correctly, but the action model, target size, or palette production is defective.

## Routes

| Stable ID | Class | Trigger signature | Expected contribution | Exact contract |
|---|---|---|---|---|
| `visual-hierarchy-and-salience` | Core | Attention lands in the wrong place or emphasis no longer matches consequence. | An attention map with grouping, focal priority, background inhibition, and rest. | [Load contract](capability-visual-hierarchy-and-salience.md) |
| `information-density-and-disclosure` | Core | Working memory is overloaded or complexity appears at the wrong moment. | A density and disclosure plan that stages necessary complexity responsibly. | [Load contract](capability-information-density-and-disclosure.md) |
| `reading-and-scanning-composition` | Core | Operational text is present but costly to scan, parse, or compare. | A rewritten reading path with typographic and semantic hierarchy. | [Load contract](capability-reading-and-scanning-composition.md) |
| `state-memory-and-recognition` | Core | Users must remember hidden state, prior steps, codes, or transformations. | Externalized state and recognition cues that reduce recall. | [Load contract](capability-state-memory-and-recognition.md) |

**Cross-domain handoffs:** Signal Spatial Composition when mass hierarchy causes the attention error; Color when value or color relations carry emphasis; Motion when temporal salience competes; Human Fit when expectation, low vision, or cognitive load changes noticing.
