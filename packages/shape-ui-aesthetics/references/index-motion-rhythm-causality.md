# Motion, Rhythm & Causality Index

**Decision question:** Does motion earn its cost by explaining cause, state, hierarchy, continuity, progress, or identity, and can it do so safely and responsively across users and target devices?

**Material signals:** Motion exists because the interface should feel lively; the causal job is unnamed; a transition obscures state; identity motion competes with task feedback; an accepted motion blocks input, risks comfort, depends on movement for meaning, or fails under target device constraints.

**Non-signals:** Purpose, spatial model, accessibility behavior, and device viability are already accepted, and the only open issue is one fixed implementation value with no effect on those decisions.

**Selection seams:** Establish the temporal job with motion purpose before evaluating an existing hypothesis through motion quality in Coherence. Route accessible responsive motion for temporal comfort, interruption, flashing, motion-dependent meaning, and device degradation. Signal Human Fit when semantic layering, readability, focus, hit testing, or redundant effect channels are independently unresolved.

**Cross-domain handoffs:** Signal Interaction when motion must repair an action-feedback loop; Information when temporal emphasis changes attention; Medium when representation determines motion; Human Fit for comfort or reduced-motion behavior; Coherence when local animation lacks a shared language.
