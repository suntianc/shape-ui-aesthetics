# Computational Medium & Embodiment Index

**Decision question:** Which computational body and intrinsic responsive composition make the intended effect, protagonist, ritual, and core relationships load-bearing under real content, control, container, input, and resource constraints?

**Material signals:** The same concept could use materially different media; the medium is chosen by fashion; effect evidence or fallback is undefined; accepted composition regimes depend on device breakpoints or fail between content, container, input, and capability conditions.

**Non-signals:** The platform, representation, and responsive relationships are accepted and the remaining issue is one local implementation parameter that cannot change medium or composition behavior.

**Selection seam:** Route medium selection when candidate computational bodies would materially change the effect or control model. Route flexible responsive composition only after composition regimes are accepted and their implementation fails to adapt between them.
