Local craft archive · evidence-led demo

Common Thread

Eight collection records are arranged by what can be verified: when an object was made, where its record places it, who is named, which materials remain, and how those materials were worked.

Follow the making. Filtering is not a cosmetic layer: each choice tightens the visible thread between material, action, maker, place, and time.

Structure studies are diagrams, not replicas. No patina, simulated wear, or symbolic interpretation is used as a shortcut to "craft."

Preparing local records
Time / 01

The record has a position.

Time is a working coordinate, not an ornamental date stamp. Choose a marker to open the corresponding record; filtering redraws the chronology around the surviving evidence.

Evidence set / 02

Material becomes method.

Each card exposes a three-part making chain. Open a record to inspect source attribution, cultural boundaries, and related methods without losing the current filter state.

Fact protocol / 03

What the archive refuses to guess.

Collection metadata can be precise and still incomplete. The interface keeps that incompleteness visible instead of turning it into atmosphere.

01 · Attribution Unknown stays unknown.

"Maker unrecorded" is a first-class value. It is searchable, filterable, and never replaced with a speculative identity.

02 · Meaning Interpretation is attributed.

This demo uses object facts from named museum records. It does not infer universal color, motif, spiritual, or community meanings.

03 · Images Absence does not erase evidence.

An unavailable image remains a named state; maker, material, technique, date, place, source, and limitations remain usable.
