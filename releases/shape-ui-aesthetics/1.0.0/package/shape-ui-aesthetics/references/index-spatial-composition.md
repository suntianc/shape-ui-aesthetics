# Spatial Composition Index

**Decision question:** Do mass, void, sequence, scale, and position create a legible whole with a deliberate center of gravity?

**Material signals:** The page lacks a convincing whole; regions compete as equal slabs; the protagonist has no spatial authority; responsive composition loses hierarchy.

**Non-signals:** The spatial organization is sound and the issue is text phrasing, color calibration, or interaction feedback alone.

**Selection seam:** Route organization when governing composition regimes or dominant/subordinate relationships are unresolved. Route weight, rhythm, and proportion when the system-wide mass or tempo target is unresolved. Signal Computational Medium only after those regimes are accepted and their implementation fails between responsive conditions; signal Coherence for a localized perceived-versus-measured optical discrepancy.

**Cross-domain handoffs:** Signal Information when attention order does not follow spatial order; Medium when depth or dimensionality is load-bearing; Interaction when organization changes action or wayfinding; Human Fit when reflow or physical reach changes the composition.
