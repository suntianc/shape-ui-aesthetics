# Color, Light & Material Index

**Decision question:** Do color, value, quantity, illumination, edge, and surface relationships support hierarchy, atmosphere, and material truth?

**Material signals:** Palette choices are isolated swatches; value structure collapses; an accent loses exclusivity; area and placement undo an otherwise plausible palette.

**Non-signals:** Mandatory semantic colors are already fixed and the open issue is navigation, copy, or geometry rather than their relational application.
